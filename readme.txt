пример интеграции 1с розница и фискальных регистраторов сервис плюс
СервисПлюс.cfe - обработчик обслуживания 1с
server service - служба сервера

СервисПлюс.cfe  формирует и отправляет post запрос(текст xml) в формате оборудования в службу сервера,
служба сервер слушает порт tcp(http), передает данные полученные через порт в ком порт, получает ответ отправляет в 1с.

СервисПлюс получает ответ(в том же соединении), парсит xml полученный от оборудования.

параметры оборудования передаются в хидере запроса 

служба сервера - многопоточная, имеет варианты логирования в консоль, в файл
